# S&P 500 Investing Data

For strict replication of Gong & Xing (2024), place the Investing.com S&P 500
historical CSV here:

```text
data/raw/sp500_investing.csv
```

Use the date range `2011-05-02` through `2023-03-31`. The loader accepts common
Investing column names such as `Date / High / Low`, `日期 / 高 / 低`, and
`日期 / 最高價 / 最低價`.
